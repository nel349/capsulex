# tradeModal.style

A comprehensive stylesheet for the TradeModal component, providing styling for various UI elements.

## Overview

- Container and layout styles for modal presentation
- Header and navigation tab styling
- Token selection row and selector styles
- Input fields and label designs
- Button and loading state styles
- Tab content and scroll container styles
- Result and error message formatting
- Share prompt modal styling for trade confirmations

## Usage

```tsx
import styles from 'path/to/components/thread/trade/tradeModal.style';

<View style={styles.modalContentContainer}>
  <Text style={styles.headerTitle}>Trade</Text>
  <TouchableOpacity style={styles.swapButton}>
    <Text style={styles.swapButtonText}>Swap</Text>
  </TouchableOpacity>
</View>
```

## Description

SelectTokenModal provides a searchable interface for selecting tokens from a list
of verified tokens fetched from Jupiter's API. It fetches the list of verified tokens
when the modal becomes visible, and allows users to search and filter the tokens.

Key functions:
- `fetchTokens()`: Fetches the list of verified tokens from Jupiter's API
- `filteredTokens`: Memoized computation of tokens filtered by search input
- `renderItem()`: Renders a single token item in the list

Features:
- Live search with filtering by symbol, name, or address
- Display of token logos with fallback for missing images
- Clean token information presentation with symbol and name
- Loading indicator during API requests
- Modal with proper open/close handling

## Example

```tsx
<SelectTokenModal
  visible={showTokenModal}
  onClose={() => setShowTokenModal(false)}
  onTokenSelected={(token) => {
    setSelectedToken(token);
    updateTradeAmount(token);
  }}
/>
```

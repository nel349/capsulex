## Description

TradeModal provides a user interface for executing token swaps using Jupiter aggregator
and sharing the trade details on the social feed. 

The component handles two main flows:
1. Swap & Share (posts a trade summary on success)
2. Pick an existing Tx signature & share it

Features:
- Token selection via integrated SelectTokenModal
- Real-time price quotes from Jupiter API
- Trade execution with wallet integration
- Custom confirmation UI for sharing trades
- Multiple tab interfaces
- Detailed error handling and loading states
- Responsive design with keyboard-avoiding behavior

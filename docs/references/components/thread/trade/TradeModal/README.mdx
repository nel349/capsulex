# TradeModal Component

The TradeModal component provides an interface for users to execute token trades on Solana and share them on the social feed.

## Overview

- Handles two main flows:
  - Swap & Share (execute token trade and post a summary)
  - Pick an existing transaction signature & share it
- Provides token selection interface with Jupiter integration
- Features interactive UI with loading states and error handling
- Includes a confirmation modal for sharing trades
- Implements Solana wallet integration for transaction signing
- Supports customizable token selection

## Usage

```tsx
import TradeModal from 'path/to/components/thread/trade/TradeModal';

<TradeModal
  visible={showTradeModal}
  onClose={() => setShowTradeModal(false)}
  currentUser={userObject}
  onPostCreated={handlePostCreated}
  initialInputToken={solToken}
  initialOutputToken={usdcToken}
  disableTabs={false}
/>
```

# Thread Components

The Thread component system provides a comprehensive set of components and utilities for rendering social media thread-like experiences in your application.

## Available Components

- **Thread.tsx** - The main thread container component
- **ThreadComposer.tsx** - Interface for creating new thread posts
- **ThreadItem.tsx** - Individual thread post component
- **ThreadAncestors.tsx** - Displays thread ancestry for nested conversations
- **ThreadEditModal.tsx** - Modal for editing existing thread posts
- **NftListingModal.tsx** - Modal for displaying and interacting with NFT listings

## Utilities and Helpers

- **thread.types.ts** - TypeScript definitions for thread components
- **thread.styles.ts** - Styling utilities for thread components
- **thread.theme.ts** - Theming configuration for thread components
- **thread.utils.ts** - Helper functions for thread functionality

## Specialized Thread Sections

The thread component system includes specialized sections for:
- Post creation and display
- NFT trading functionality
- Retweet capabilities
- Followers/Following user management

## Usage

Import individual components from the thread module:

```tsx
import { Thread, ThreadItem, ThreadComposer } from 'path/to/components/thread';
```

See individual component documentation for detailed usage instructions.

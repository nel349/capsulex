## Component

## Description

PostBody handles the rendering of different types of content sections in a post,
including text, images, videos, polls, trades, and NFT listings. It supports
multiple sections per post and delegates rendering to specialized section components.

The component is wrapped in React.memo with a custom comparison function to optimize
rendering performance by preventing unnecessary re-renders.

## Features

- Multiple content section support
- Section type-specific rendering
- Customizable styling through theme and style overrides
- Responsive layout
- Performance optimized with memoization
- External refresh trigger for trade charts

## Supported Section Types

- TEXT_ONLY: Plain text content
- TEXT_IMAGE: Text with an image
- TEXT_VIDEO: Text with a video
- TEXT_TRADE: Trade information
- POLL: Poll data
- NFT_LISTING: NFT listing details

# PostBody Component

The PostBody component renders the content sections of a thread post, supporting multiple types of content including text, images, videos, polls, trades, and NFT listings.

## Overview

- Renders multiple content sections within a post
- Delegates rendering to specialized section components
- Supports customizable styling
- Optimized with React.memo for better performance
- Configurable theme and style overrides

## Section Types

The component can render the following section types:
- Text only content
- Text with images
- Text with videos
- Trading information
- Polls
- NFT listings

## Usage

```tsx
import PostBody from 'path/to/components/thread/post/PostBody';

<PostBody 
  post={postData}
  themeOverrides={customTheme}
  styleOverrides={customStyles}
  externalRefreshTrigger={refreshCounter}

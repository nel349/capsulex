# PostFooter Component

The PostFooter component renders the bottom section of a thread post, providing various interaction controls and displaying engagement metrics.

## Overview

- Displays comment, retweet, and reaction counts
- Provides interactive buttons for user engagement
- Features an emoji reaction selector
- Includes bookmark functionality
- Integrates with Redux for state management
- Opens a retweet modal for sharing content
- Customizable through theme and style overrides

## Usage

```tsx
import PostFooter from 'path/to/components/thread/post/PostFooter';

<PostFooter 
  post={postData}
  onPressComment={handleComment}
  themeOverrides={customTheme}
  styleOverrides={customStyles}
/>
```

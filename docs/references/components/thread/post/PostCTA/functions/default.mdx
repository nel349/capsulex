## Component Details

### Key Functions

1. **getPostSectionType**
   ```typescript
   function getPostSectionType(post: ThreadPost): 'trade' | 'nft' | 'collection' | null
   ```

   Determines the type of CTA to display based on the post's sections.

2. **getTradeData**
   ```typescript
   function getTradeData(post: ThreadPost): TradeData | null
   ```
   Extracts trade data from a post's TEXT_TRADE section.

3. **handleBuyListedNft**
   ```typescript
   async function handleBuyListedNft(): Promise<void>
   ```
   Handles the NFT purchase process including transaction creation and signing.

4. **handleBuyCollectionFloor**
   ```typescript
   async function handleBuyCollectionFloor(): Promise<void>
   ```
   Handles the purchase of floor NFT from a collection.

### Features

- Automatic CTA type detection
- Integrated wallet connection checks
- Transaction status feedback
- Error handling and user notifications
- Loading state management
- Modal-based interfaces for trades and confirmations

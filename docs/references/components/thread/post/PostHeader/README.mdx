# PostHeader Component

The PostHeader component renders the top section of a thread post, displaying user information and post metadata.

## Overview

- Displays the user's avatar, username, and verification status
- Shows post creation time
- Provides a dropdown menu for post actions (edit/delete)
- Supports user interaction via avatar/username tapping
- Customizable through theme and style overrides

## Usage

```tsx
import PostHeader from 'path/to/components/thread/post/PostHeader';

<PostHeader 
  post={postData}
  onDeletePost={handleDelete}
  onEditPost={handleEdit}
  onPressUser={handleUserPress}
  themeOverrides={customTheme}
  styleOverrides={customStyles}
/>
```

## Component

## Description

CoinDetailTopSection provides a container that includes:
- A search bar for switching coins
- Display of the current coin's price, change, and chart
- Market stats (Market Cap, FDV, etc.)
- Example tweet data
- Buttons (Swap, Send)

## Example

```tsx
<CoinDetailTopSection
  tweetData={[
    {
      username: 'Alice',
      handle: '@alice',
      time: '2h',
      tweetContent: 'Check out $SEND!',
      quoteCount: 5,
      retweetCount: 10,
      reactionCount: 100,
      avatar: {uri: 'https://example.com/avatar.png'},
    },
  ]}
/>
```

`Element`

## Component

## Description

ProfileInfo is used to display key profile details for a given user:
- **Avatar** (with editable option if it’s the user's own profile)
- **Username** and short wallet handle
- **Bio text**
- **Follow/Unfollow logic** (if viewing another’s profile)
- **Optional Buy and Perks components**

## Example

```tsx
<ProfileInfo
  profilePicUrl="https://example.com/avatar.png"
  username="Alice"
  userWallet="xxxxxx..."
  isOwnProfile={false}
  amIFollowing={true}
  areTheyFollowingMe={false}
  onFollowPress={() => {}}
  onUnfollowPress={() => {}}
/>
```

## Component

## Description

`Profile` is a higher-level component that manages the user's overall profile data and view. It handles:
- Fetching additional profile data (username, avatar)
- Managing followers/following
- Merging post data
- Merging NFT data (or default fetch)
- Rendering `ProfileView` to display the final UI

**Features**:
- Optional local or remote image picking for avatar
- NFT-based avatar selection
- Username editing
- Follow/unfollow logic

## Example

```tsx
<Profile
  isOwnProfile={true}
  user={{ address: 'YourWalletAddressHere' }}
/>
```

# components/profile.style

This file exports style objects used in `Profile.tsx` for modals, confirmations, etc. It includes:

- `styles.container`
- `modalStyles.overlay`, `modalStyles.optionContainer`, ...
- `confirmModalStyles.overlay`, ...
- `inlineConfirmStyles.container`, ...
- `editNameModalStyles.overlay`, ...

They help control layout for picking avatars, editing the username, and confirming changes.

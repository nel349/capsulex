## Component

## Description

`ProfileView` is a presentational component that arranges:
- The top portion (ProfileInfo)
- The bottom portion (SwipeTabs for posts, collectibles, actions)

This is meant to be a lightweight wrapper that you can embed in your own screens if you already have user data, or as part of the higher-level `Profile` component.

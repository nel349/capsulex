# Introduction to Solana App Kit

Solana App Kit is a React Native library providing ready-to-use components, services, and utilities for building Solana-based web3 applications. It offers modular, customizable implementations for authentication, social features, token transactions, and NFT management.

The library is designed to be easily integrated into new or existing React Native applications, with flexible configuration options to adapt to your specific needs.

This guide will help you get up and running with Solana App Kit quickly.

## Installation

```bash
npx install solana-app-kit
```
